# ImplementAI Labs - Strategic Architect Rules

## Token Efficiency Commands
- Be extremely brief. Use prose only for complex strategy.
- NEVER use conversational filler like "I'd be happy to help" or "I have analyzed your files."
- Use Markdown tables for metrics and checklists for tasks.
- Provide code snippets only for the specific lines being changed.

- Tools: MuAPI for media, Groq/9router for LLMs.

## Operational Protocol
- Do not read subdirectories unless a task requires it.
