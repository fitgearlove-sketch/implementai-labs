# Proposal Template - ImplementAI Labs

## Purpose & Usage Notes

**When to send:** Within 48 hours of the Discovery/Diagnostic Call (per Email 7 in `EMAIL-TEMPLATES.md`)

**Format:** Google Docs → Export to PDF (or Notion/PandaDoc if using proposal software)

**Validity:** 14 days from send date (creates gentle urgency)

**Goal:** Convert Discovery Call insights into a clear, ROI-justified, low-friction "yes"

## Proposal Best Practices

**Before Sending:**
- [ ] Confirm all placeholder variables are filled in (no {brackets} remain)
- [ ] Verify pricing matches package tier discussed on call
- [ ] Double-check math on ROI calculations
- [ ] Proofread for agency name/details accuracy
- [ ] Export as branded PDF (not raw Google Doc link)

**Sending:**
- Send within 48 hours of Discovery Call (use Email 7 template)
- CC nobody unless explicitly requested (keep it personal)
- Follow up on Day 3 if no response (use Email 8 template)

**Tracking:**
- Log proposal sent date in CRM
- Track open/view if using PandaDoc or similar
- Set reminder for Day 3 and Day 7 follow-ups

**Common Objections & Responses:**
| Objection | Response Approach |
|-----------|-------------------|
| "Too expensive" | Reframe around Cost of Inaction, not price |
| "Need to think about it" | Ask what specific concern needs addressing |
| "Need buy-in from partner/team" | Offer to join a call with decision-makers |
| "Not the right timing" | Ask what would need to change for timing to work |
