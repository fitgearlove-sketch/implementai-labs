## Copy Guidelines Reference (from Content Strategy)

- **Tone:** Confident, analytical, ROI-focused — avoid overhyped AI promises
- **Show, don't tell:** Use specific numbers ($, hours) over vague claims
- **Niche specificity everywhere:** Every page should make clear this is
  built for performance marketing + e-commerce agencies, not generic
  "AI for business" positioning
- **CTA hierarchy:** Free Audit is always the primary CTA — don't compete
  with it using multiple conflicting CTAs on the same page
